# Digital Geosciences Platform

Live site: https://digitalgeosciences.com

## Overview
Static website for the Digital Geosciences platform. Content in `public/` powers the pages built from `src/`.

## Quick Start
1. Install deps: `npm install` (or `pnpm install`).
2. Run locally: `npm run dev`.
3. Build for production: `npm run build`.

## Data Sources
- `public/data/projects.json` — project listings.
- `public/data/collaborators.json` — collaborator profiles.

Update the JSON, rebuild, then redeploy the `dist/` output to publish changes.
